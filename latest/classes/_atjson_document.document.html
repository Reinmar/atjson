where

  • where(filter: object | function): AnnotationCollection
    • Defined in document/src/index.ts:105

    A simple example of this is transforming a document parsed from an HTML document into a common format:

    html.where({ type: 'h1' }).set({ type: 'heading', attributes: { level: 1 }});

    When the attribute for h1 is added to the document, it will be swapped out for a heading with a level of 1.

    Other options available are renaming variables:

    html.where({ type: 'img' }).set({ 'attributes.src': 'attributes.url' });
